# ProfileSmith 1.0

- Rebuilt a native macOS provisioning profile manager in Swift with SQLite + GRDB for scalable indexing and search.
- Fixed launch behavior so the main window reliably appears on first open and reopen.
- Added built-in Finder Quick Look extensions for provisioning profiles, IPAs, XCArchives, apps, and app extensions.
- Kept drag-and-drop import, archive inspection, export, trash/delete, and filename beautification in a faster UI.
- Added regression, parser, database, scanner, file operation, and UI automation coverage for the first public release.
