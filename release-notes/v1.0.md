# ProfileSmith 1.0

- Built a new macOS provisioning profile manager from the ground up in Swift.
- Indexed profiles with SQLite + GRDB for fast search, filtering, and sorting at scale.
- Added archive and bundle inspection for apps, IPAs, XCArchives, and extensions.
- Included Quick Look integration, drag and drop import, safe export, and filename beautification.
- Added a status bar menu and Sparkle-based update support.
- Covered parser, database, scanner, file operations, and UI controller behavior with automated tests.
